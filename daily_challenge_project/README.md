# Daily Challenge Web Application

## Project Description
Daily Challenge is a web application that helps users build good habits by completing small daily challenges. Users can select a category, complete daily tasks, track their progress, and write reflections about their activities.

## Features
- User Registration and Login
- Category Based Challenge Selection
- Daily Challenge System
- Complete or Skip Challenges
- History Tracking
- Reflection Journal
- Progress Statistics Dashboard

## Technologies Used
- Python
- Flask Framework
- SQLite Database
- HTML
- CSS

## How to Run the Project

1. Install Python
2. Install Flask

pip install flask

3. Run the application

python app.py

4. Open browser and go to:

http://127.0.0.1:5000
